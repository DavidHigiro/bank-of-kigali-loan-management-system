# bank-of-kigali-loan-management-system
A Java OOP-based Loan Management System for Bank of Kigali demonstrating abstraction, inheritance, polymorphism, interfaces, and Docker deployment.
## Overview
This project simulates a real-world Loan Management System for Bank of Kigali. It is built using core Object-Oriented Programming (OOP) principles and demonstrates clean architecture, input validation, and scalable design.

The system allows users to:
- Create and manage different loan types
- Validate loan and customer details
- Process repayments
- Generate loan and payment reports
